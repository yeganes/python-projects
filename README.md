# Personal Expense Analyzer

A beginner-friendly Python project that tracks expenses and performs simple data analysis.

## Features
- User enters multiple expenses
- Stores data in a dictionary
- Calculates:
  - Total spending
  - Average expense
  - Highest expense (name + amount)
  - Lowest expense (name + amount)

## What I Learned
- Using for loops
- Dictionaries to pair names with values
- List operations
- `min()`, `max()`, `sum()`
- Basic data analysis logic
